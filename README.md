# Titanic Classification Project

## 📌 Overview
This project predicts passenger survival on the Titanic dataset using **Machine Learning models**.  
It demonstrates end-to-end workflow: data preprocessing, model training, evaluation, and visualization.

## ⚙️ Tech Stack
- Python (Anaconda environment `ai_env`)
- Pandas, NumPy
- Scikit-learn (Logistic Regression, Decision Tree)
- Matplotlib, Seaborn
- Jupyter Notebook

## 🚀 Steps
1. Load and preprocess Titanic dataset (handle missing values, encode categorical features).
2. Train models:
   - Logistic Regression
   - Decision Tree
3. Evaluate performance using accuracy, confusion matrix, classification report.
4. Visualize results with heatmaps and plots.
